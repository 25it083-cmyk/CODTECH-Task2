OVERVIEW OF THE PROJECT

Introduction

Movie Search Engine is a web-based application used to search and display movie details online. Users can enter a movie name in the search box and get movie information instantly. The application provides a simple, attractive, and user-friendly interface.

Objective

The main objective of this project is to help users search movies easily and view important details such as movie poster, title, release year, and movie type. It also helps beginners understand API integration in web development.

Technologies Used

The project is developed using HTML, CSS, and JavaScript. HTML is used to create the webpage structure, CSS is used for styling and designing the interface, and JavaScript is used to fetch movie data dynamically from the OMDb API.

Working of the Project

When the user enters a movie name and clicks the search button, JavaScript sends a request to the OMDb API. The API returns movie information, and the details are displayed on the webpage in the form of movie cards with posters and text information.

Features

The project includes a responsive design, movie poster display, real-time movie search, attractive pink theme, and dynamic content loading. It also shows error messages when the movie is not found.

Conclusion

Movie Search Engine is a simple and interactive web application that helps users search movies quickly. This project improves knowledge in frontend development, API usage, and dynamic webpage creation.
